# Sailing Shopper

A RuneLite plugin to help manage shopping lists for Construction and Sailing.

## Features

*   **Add to Shopping List**: Context menu option on Construction and Sailing guides to add materials to your shopping list.
*   **Smart Parsing**: Automatically parses item names and quantities (e.g., "Oak plank x 8").
*   **Simple UI**: View your shopping list in the sidebar and remove items as needed.

## Usage

1.  Open the Construction or Sailing skill guide in-game.
2.  Right-click on a material requirement.
3.  Select "Add materials to Shopping List".
4.  View your list in the RuneLite side panel.
